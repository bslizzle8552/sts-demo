# STS Demo

Local React/Vite simulator for a Sullair STS-style compressor controller.

## Run Locally

```powershell
npm.cmd install
npm.cmd run dev
```

Open `http://127.0.0.1:5173`.

## Checks

```powershell
npm.cmd run lint
npm.cmd run build
```
